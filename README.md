## 🛠 Requirements

Install Python and NumPy.

```bash
pip install numpy
```

To open the notebooks:

```bash
pip install notebook
```

Then run:

```bash
jupyter notebook
```

---

## ▶️ How to Use

1. Clone the repository.

```bash
git clone https://github.com/jeetgondaliya2-prog/Numpy.git
```

2. Open the project folder.

```bash
cd Numpy
```

3. Start Jupyter Notebook.

```bash
jupyter notebook
```

4. Open any notebook (`Ep-1.ipynb`, `Ep-2.ipynb`, etc.) and start learning.
